# Traffic Light Simulation ‚Äì 3 Motors, Standard Sequence

This project represents a classic control and power circuit built in **CADe SIMU** using **3 motors** to simulate the output of standard red, yellow, and green traffic lights.

Each motor represents one lamp, and the sequence is controlled through timers and relay logic.

---

## üö¶ Sequence Logic

The control sequence follows a standard timing-based cycle:

1. **Red**
2. **Yellow + Red**
3. **Green**
4. **Yellow**
5. **Red**

All lights operate with **equal timer durations** for simplicity and clarity.
