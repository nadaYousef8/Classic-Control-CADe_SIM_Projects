# CADe SIMU ‚Äì Classic Control Projects

This repository contains a collection of classic electrical control circuits designed and simulated using **CADe SIMU**.  
Each project includes:
- Power circuit diagram

- Sequence description
- Timers & relay logic

The projects demonstrate fundamental industrial control concepts such as forward/reverse motor operation, interlocking, protection components, and timing-based sequencing.
